## Setup

MacOS
```
brew install elm
```

Linux ?
```
apt-get install elm
```

```
npm i -g elm-format
npm i -g create-elm-app
```

## Development

Start development server
```
elm-app start
```

http://localhost:3000/

## Tests

Running tests
```
elm-app test
```

## Production build

```
elm-app build
```


## Adding dependecies

For Elm
```
elm install elm/svg
```

For JavaScript
```
npm install d3-dagre
```
